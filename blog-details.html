Pengalaman Pertama Pakai GitHub

Beatrice

Siswa PKL SMK
Jul 10, 2025 • 5 Comments

Di tempat PKL-ku, semua projek wajib dipush ke GitHub. Awalnya aku bingung banget, apalagi belum pernah pakai Git sebelumnya.

Tapi setelah dipandu mentor dan mulai belajar perintah dasar seperti git init, git add ., git commit, dan git push origin main, aku mulai paham cara kerja version control. Ternyata seru banget, dan sangat ngebantu kalau kerja tim.

Apa itu GitHub?

GitHub adalah platform untuk menyimpan dan berkolaborasi dalam proyek kode secara online. Jadi kita bisa bekerja bareng teman, melihat perubahan kode, dan menghindari konflik saat revisi.

  • Menyimpan proyek secara cloud
  • Membantu tracking versi kode
  • Mendukung kolaborasi tim
  • Bisa digunakan buat portofolio

"Aku jadi makin semangat belajar ngoding karena bisa lihat progresku langsung di GitHub."

– Beatrice

Kenapa Siswa SMK Harus Belajar Git?

Di dunia kerja, Git dan GitHub adalah tools yang umum digunakan. Kalau kita udah terbiasa dari SMK, nanti pas masuk industri nggak kaget lagi. Selain itu, GitHub juga bisa jadi tempat showcase portofolio buat masuk kuliah atau cari kerja.

Semoga pengalaman pertamaku ini bisa jadi motivasi kamu buat mulai belajar Git ya!
